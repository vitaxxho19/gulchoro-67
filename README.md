# Счётчик Частоты Слов

## Описание
Счётчик частоты слов — это проект, который позволяет пользователю вводить текст и получать количество вхождений каждого слова в этом тексте. Это полезный инструмент для анализа текста, исследования частотности слов и выполнения различных лингвистических зад��ч.

## Инструкции по Использованию
1. Склонируйте репозиторий на свою машину:  
   ```bash
   git clone https://github.com/vitaxxho19/gulchoro-67
   ```  
2. Перейдите в директорию проекта:  
   ```bash
   cd gulchoro-67
   ```  
3. Запустите программу:  
   ```bash
   python word_frequency_counter.py
   ```
4. Введите текст, который вы хотите проанализировать.

## Примеры
- Ввод: "Привет мир мир!"
- Вывод:  
  - мир: 2  
  - Привет: 1
  - !: 1

## Особенности
- Подсчет частоты для любой длины текста.  
- Игнорирование регистра (примеры: "Слово" и "слово" считаются одинаковыми).  
- Поддержка различных символов и пунктуации.  
- Легко расширяемая архитектура для добавления новых функций.  
- Возможность вывода частоты в виде графиков и таблиц.
